

Simple PHP MVC Framework

A simple PHP MVC framework with user authentication and Tailwind CSS styling. Includes Home, Login, Register, and Dashboard pages with a MySQL database backend.

Features

MVC architecture (Models, Views, Controllers)

User authentication (register, login, logout)

Protected dashboard for logged-in users

Tailwind CSS for modern and responsive UI

Simple routing for easy navigation

Requirements

PHP 7.4 or higher

MySQL (XAMPP or similar)

Apache web server
